## CN の欠落が分かる表記を追加する（CSR プレビュー・CN 検証機能）

### 経緯

クライアント証明書署名時に、CSR に Common Name (CN)が含まれていないことに気づかずに署名してしまうケースがあった。CN が欠落している CSR の署名を防止する仕組みが必要であった。

### 対応方針

- CSR 入力時にサブジェクト情報のプレビュー機能を追加し、CN の有無に応じて署名可否を制御する。

### 実装内容

**フロントエンド**

![image.png](./image.png)

- CSR の内容を署名前に確認できるプレビュー表示を追加
- Common Name を含む主要なサブジェクト情報を画面上で確認できるように対応
- 値が入っていない項目も判別しやすい表示に変更
- Common Name の有無や CSR の妥当性に応じて、署名可否を画面上で分かりやすく案内するよう改善

### 検証済みの内容

- CSR を入力後、プレビューボタンを押下するとサブジェクト情報がプレビュー表示されること
- プレビュー画面に 7 項目（Common Name, Country Name 等）が表示されること
- 値が存在しないフィールドは「-----」で表示されること
- （CN 有り）緑色のアラートが表示され、署名ボタンが有効であること
- （CN 有り）署名ボタン押下で署名済み証明書ファイルがダウンロードされること
- （CN 無し）黄色のアラートが表示され、署名ボタンがグレーアウトされること
- （不正な CSR）黄色のアラートが表示され、署名ボタンがグレーアウトされること
- 【リグレッション】正常な CSR での署名が従来通り動作すること
