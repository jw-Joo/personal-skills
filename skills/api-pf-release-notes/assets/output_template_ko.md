## CSR 미리보기 및 CN 검증 기능에 CN 누락을 알 수 있는 표시 추가

### 배경

클라이언트 인증서 서명 시 CSR에 Common Name(CN)이 포함되어 있지 않은 것을 알아차리지 못한 채 서명하는 경우가 있었다. CN이 누락된 CSR의 서명을 방지할 수 있는 구조가 필요했다.

### 대응 방침

- CSR 입력 시 주체 정보를 미리 확인할 수 있는 기능을 추가하고, CN 존재 여부에 따라 서명 가능 여부를 제어한다.

### 구현 내용

#### 프론트엔드

![image.png](./image.png)

- CSR 내용을 서명 전에 확인할 수 있는 미리보기 표시를 추가
- Common Name을 포함한 주요 주체 정보를 화면에서 확인할 수 있도록 대응
- 값이 없는 항목도 식별하기 쉽도록 표시를 변경
- Common Name 존재 여부와 CSR 유효성에 따라 서명 가능 여부를 화면에서 알기 쉽게 안내하도록 개선

### 검증한 내용

- CSR 입력 후 미리보기 버튼을 누르면 주체 정보가 미리보기로 표시되는 것
- 미리보기 화면에 7개 항목(Common Name, Country Name 등)이 표시되는 것
- 값이 없는 필드는 「-----」로 표시되는 것
- (CN 있음) 녹색 알림이 표시되고 서명 버튼이 활성화되는 것
- (CN 있음) 서명 버튼을 누르면 서명된 인증서 파일이 다운로드되는 것
- (CN 없음) 노란색 알림이 표시되고 서명 버튼이 비활성화되는 것
- (잘못된 CSR) 노란색 알림이 표시되고 서명 버튼이 비활성화되는 것
- [회귀 테스트] 정상 CSR의 서명이 기존처럼 동작하는 것
